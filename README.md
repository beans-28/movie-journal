Installation Guide (Windows):

1. Download and Install XAMPP (apachefriends.org)

2. Open the XAMPP Control Panel and click Start for both Apache and MySQL.

3. Download the ZIP of the repository from GitHub and extract it.

4. Move the extracted folder into XAMPP directory: C:\xampp\htdocs\movie-journal.

5. Go to http://localhost/phpmyadmin in your browser and create a new database named "movie_journal".

6. Select that new database, click the Import tab, upload the database.sql file from the project folder.

7. Open browser then go to http://localhost/movie-journal to run the app.
